# HIP Lumi tools

Tools to get luminosities, pileup profiles and prescales that can be passed to other software.

## Setup

All scripts here are intended to be run on Lxplus with CMSSW and brilcalc. The initial setup can be done with the `create_environment.sh` script.

```
source create_environment.sh
```

and then in later sessions activated with

```
source activate_environment.sh
```
